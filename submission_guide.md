# SmartStock AI: Project Submission Guide

## 1. Core Architecture: "The Speed of Light"
Use this hierarchy in your presentation/video to explain the system:

| Layer | Component | Function |
| :--- | :--- | :--- |
| **Intelligence** | Gemma 4 / Google AI Studio | The "Brain" that processes inventory data, recognizes trends, generates restock recommendations, and streams predictions back through the Antigravity bridge. |
| **Orchestration** | Antigravity Framework | The "Nervous System" that cleans messy data, routes API calls, and bridges the async connection. |
| **Interface** | Responsive Web / PWA | The "Face" where users interact with the dashboard and predictor tools. |
